Sequences

A sequence is a list of numbers that follows a rule. You will learn how to spot the rule, write the nth term, and use sequences in real-life problems. We start with simple patterns and build up to sums and harder types.

1 Term-to-Term Rules All Exam Boards Year 8

Example: The list 3, 7, 11, 15, … goes up by 4 each time. The term-to-term rule is: add 4.

A sequence is an ordered list of numbers. The easiest way to describe it is the term-to-term rule: what you do to one term to get the next.

  • Arithmetic (linear): add or subtract the same number each time. That number is the common difference \(d\).
  • Geometric: multiply by the same number each time. That number is the common ratio \(r\).
  • Other: Fibonacci (add the last two), square numbers, triangle numbers, and more.

Sequences worth knowing: square numbers \((1, 4, 9, 16, \ldots)\), triangle numbers \((1, 3, 6, 10, \ldots)\), Fibonacci \((1, 1, 2, 3, 5, 8, \ldots)\).

Quick Check

For 2, 6, 18, 54, …, what is the term-to-term rule?

2 The \(n\)th Term of an Arithmetic Sequence All Exam Boards Year 9

Example: For 5, 8, 11, 14, … you add 3 each time, so the nth term is 3n + 2. Check: when \(n = 1\), \(3(1) + 2 = 5\) ✓

The nth term tells you any term straight away — you do not need to write out all the terms before it.

How to find it:

  1. Find \(d\) = next term − previous term (e.g. 8 − 5 = 3).
  2. Start with \(dn\) (here, \(3n\)).
  3. Compare with the first term: \(3 \times 1 = 3\), but we need 5, so add 2.

For first term \(a\) and common difference \(d\):

\[u_n = a + (n-1)d \quad \Longrightarrow \quad u_n = dn + (a - d)\]

So the nth term is always a linear expression like \(3n + 2\).

Always check: plug in \(n = 1\) and one other value from the list.

Is a number in the sequence? Set the nth term equal to that number and solve for \(n\). If \(n\) is a positive whole number, yes — otherwise no.

Quick Check

What is the nth term of 7, 10, 13, 16, …?

3 Special Sequences All Exam Boards Year 9

Example: Triangle numbers build up like rows of dots: 1, then 1+2 = 3, then 1+2+3 = 6. The 4th triangle number is 10.

Some patterns come up so often that you should recognise them quickly:

Sequence First terms \(n\)th term
Square numbers 1, 4, 9, 16, 25, ... \(n^2\)
Cube numbers 1, 8, 27, 64, 125, ... \(n^3\)
Triangle numbers 1, 3, 6, 10, 15, ... \(\dfrac{n(n+1)}{2}\)
Powers of 2 2, 4, 8, 16, 32, ... \(2^n\)
Fibonacci 1, 1, 2, 3, 5, 8, 13, ... No simple closed form

4 Arithmetic Series — Sum of Terms Higher Skill Year 10

Example: Add the first 5 positive whole numbers: \(1 + 2 + 3 + 4 + 5 = 15\). You can also use the formula with \(n = 5\), \(a = 1\), \(l = 5\): \(S_5 = \dfrac{5(1+5)}{2} = 15\) ✓

An arithmetic series means you add up the terms of an arithmetic sequence.

\[S_n = \frac{n(a+l)}{2} \qquad \text{or} \qquad S_n = \frac{n}{2}(2a + (n-1)d)\]

\(a\) = first term, \(l\) = last term, \(n\) = how many terms, \(d\) = common difference.

Why it works (Gauss's trick): Write the sum forwards and backwards, then add:

\(S = 1 + 2 + 3 + \cdots + n\)

\(S = n + (n-1) + \cdots + 1\)

\(2S = n(n+1)\), so \(S = \dfrac{n(n+1)}{2}\).

Another example: Find \(S_{20}\) for 3, 7, 11, … (\(a=3\), \(d=4\), \(n=20\)).

\(S_{20} = \dfrac{20}{2}(2 \times 3 + 19 \times 4) = 10(6 + 76) = 10 \times 82 = \mathbf{820}\)

Quick Check

Find the sum of the first 10 positive integers (\(1 + 2 + \cdots + 10\)).

5 Quadratic Sequences Higher Skill Year 10

Example: For 3, 8, 15, 24, 35, … the first differences are 5, 7, 9, 11 — they go up by 2 each time. That constant second difference of 2 tells you the nth term starts with \(n^2\).

A quadratic sequence has a constant second difference. Its nth term looks like \(an^2 + bn + c\).

Steps to find the nth term:

  1. Work out the first differences (subtract neighbours).
  2. Work out the second differences.
  3. \(a = \dfrac{\text{second difference}}{2}\) — that gives the \(n^2\) part.
  4. Subtract \(an^2\) from each term; what is left is linear — find its nth term.
  5. Add the \(n^2\) part and the linear part together.

Quick Check

A sequence has constant second differences. What type is it?

6 Geometric Series and Sum to Infinity Higher Skill Year 11

Example: \(8 + 4 + 2 + 1 + \frac{1}{2} + \cdots\) Each term is half the last (\(r = \frac{1}{2}\)). The terms shrink towards zero, and the total approaches 16: \(S_\infty = \dfrac{8}{1 - \frac{1}{2}} = 16\).

For a geometric sequence with first term \(a\) and ratio \(r\):

  • \(n\)th term: \(u_n = ar^{n-1}\)
  • Sum of first \(n\) terms: \(S_n = \dfrac{a(r^n - 1)}{r - 1}\) (when \(r \neq 1\))
  • Sum to infinity (only if \(|r| < 1\)): \(S_\infty = \dfrac{a}{1-r}\)

When \(|r| < 1\), each term gets smaller. The total settles on a limit — we say the series converges. That is how recurring decimals link to fractions.

Recurring decimals: \(0.\overline{3} = \frac{3}{10} + \frac{3}{100} + \cdots = \dfrac{3/10}{1 - 1/10} = \dfrac{3}{9} = \dfrac{1}{3}\).

Quick Check

When can you use \(S_\infty = \dfrac{a}{1-r}\)?

7 Sequences in Context (Word Problems) Problem Solving Year 10

Example: A plumber charges £50 to visit plus £30 per hour. Totals for 1h, 2h, 3h are £80, £110, £140, … — you add £30 each time, so this is arithmetic. After 8 hours: \(30(8) + 50 = £290\).

In word problems, ask yourself:

  • Fixed amount added each time? → arithmetic sequence.
  • Fixed multiplier or percentage each time? → geometric sequence.

Tip: Spot whether the situation grows by a fixed amount (arithmetic) or a fixed multiplier (geometric) before you pick a formula.

Quick Check

A population doubles every year. What type of sequence is this?
