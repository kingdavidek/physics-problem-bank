Converting Between Fractions, Decimals and Percentages

AQA GCSE Maths — Number

Three ways of writing the same value

Fractions, decimals and percentages all represent parts of a whole. For example, 1/2 = 0.5 = 50%, so the main skill is converting accurately between the three forms.

Decimal to percentage

To convert a decimal to a percentage, multiply by 100. This is equivalent to moving the decimal point two places to the right, then adding the percentage sign.

  • 0.29 = 29%
  • 0.375 = 37.5%
  • 1.25 = 125%

Percentage to decimal

To convert a percentage to a decimal, divide by 100. This means moving the decimal point two places to the left.

  • 18% = 0.18
  • 4% = 0.04
  • 125% = 1.25

Decimal to fraction

Use place value to write the decimal as a fraction over 10, 100, 1000 and so on, then simplify if possible.

  • 0.3 = 3/10
  • 0.25 = 25/100 = 1/4
  • 0.12 = 12/100 = 3/25

Fraction to decimal

Convert a fraction to a decimal by dividing the numerator by the denominator.

  • 3/8 = 3 ÷ 8 = 0.375
  • 1/4 = 0.25
  • 2/5 = 0.4

Percentage to fraction

Write the percentage over 100, then simplify.

  • 65% = 65/100 = 13/20
  • 23% = 23/100
  • 150% = 150/100 = 3/2

Fraction to percentage

First convert the fraction to a decimal, then multiply by 100.

  • 3/8 = 0.375 = 37.5%
  • 1/5 = 0.2 = 20%
  • 3/4 = 0.75 = 75%

Recurring decimals

Some decimals repeat forever, like 0.333... or 0.181818.... These are called recurring decimals, and at Higher tier you may be asked to convert them into fractions using algebra.

Quick reference

  • Decimal → Percentage: multiply by 100.
  • Percentage → Decimal: divide by 100.
  • Decimal → Fraction: use place value, then simplify.
  • Fraction → Decimal: numerator ÷ denominator.
  • Percentage → Fraction: write over 100, then simplify.
