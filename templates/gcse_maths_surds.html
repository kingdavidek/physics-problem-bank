Surds

AQA GCSE Higher — Number (N8)

What is a Surd?

A surd is an irrational root that cannot be simplified to a whole number or fraction. It is left in square root form to give an exact answer.

  • √4 = 2 — not a surd (it simplifies to a whole number)
  • √9 = 3 — not a surd
  • √2, √3, √5, √7 — surds (cannot be simplified to a rational number)

Surds appear whenever an exam question asks for an exact answer — a decimal approximation is not acceptable.

The Two Key Rules

All surd manipulation comes from these two rules:

Rule Example
Multiplication: √a × √b = √(ab) √3 × √5 = √15
Division: √a ÷ √b = √(a/b) √20 ÷ √5 = √4 = 2

Also note: √a × √a = a — a surd multiplied by itself gives the number inside.

Simplifying Surds

Find the largest square factor of the number under the root, then split using the multiplication rule.

Example: Simplify √48

  1. Find the largest square factor of 48: that's 16 (since 16 × 3 = 48)
  2. √48 = √(16 × 3) = √16 × √3 = 4√3

Example: Write 3√12 in the form k√3

  1. √12 = √(4 × 3) = 2√3
  2. 3√12 = 3 × 2√3 = 6√3

⚠ Always use the LARGEST square factor. If you use a smaller one (e.g. √48 = √(4×12) = 2√12) you will need to simplify again.

Adding and Subtracting Surds

You can only add or subtract surds that have the same number under the root — just like collecting like terms in algebra.

  • 3√5 + 2√5 = 5√5
  • 7√2 − 3√2 = 4√2
  • √3 + √12 = √3 + 2√3 = 3√3 (simplify √12 first!)

Expanding Brackets with Surds

Use the same FOIL / grid method as with algebraic expressions.

Example: Expand (2 + √3)(2 − √3)

  • First: 2 × 2 = 4
  • Outer: 2 × (−√3) = −2√3
  • Inner: √3 × 2 = +2√3
  • Last: √3 × (−√3) = −3
  • Total: 4 − 2√3 + 2√3 − 3 = 1

Notice the pattern: (a + √b)(a − √b) = a² − b. This always gives a rational result — this is the key to rationalising.

Example: Expand (3 + √5)²

  • = (3 + √5)(3 + √5)
  • = 9 + 3√5 + 3√5 + 5
  • = 14 + 6√5

Rationalising the Denominator

A fraction with a surd on the bottom must be rationalised — the surd is moved to the numerator.

Simple denominator: √a

Multiply top and bottom by √a.

Example: Rationalise 6/√3

= (6 × √3) / (√3 × √3) = 6√3 / 3 = 2√3

Compound denominator: a + √b or a − √b

Multiply top and bottom by the conjugate (change the sign in the middle).

Example: Rationalise 2 / (1 + √5)

  • Conjugate of (1 + √5) is (1 − √5)
  • Numerator: 2(1 − √5) = 2 − 2√5
  • Denominator: (1 + √5)(1 − √5) = 1 − 5 = −4
  • Result: (2 − 2√5) / (−4) = (√5 − 1) / 2
