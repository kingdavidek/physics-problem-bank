1 Function machines All Exam Boards Year 7

Before we write f(x), most students meet functions as a function machine: a number goes in, a rule is applied, and exactly one number comes out.

  • The input is the starting value.
  • The machine tells you what to do (e.g. multiply by 2, then add 3).
  • The output is the final answer.

Apply each operation in order, working from left to right through the machine.

Example: Input 4, machine “× 2 then + 3”:

  • 4 × 2 = 8
  • 8 + 3 = 11
  • Output = 11

The same rule written with function notation is f(x) = 2x + 3 and f(4) = 11.

Each box is one step. Work through the machine in order.

Working backwards: if the output is 11 and the machine is “× 2 then + 3”, undo the steps in reverse:

  • Subtract 3: 11 − 3 = 8
  • Divide by 2: 8 ÷ 2 = 4
  • So the input was 4.

This “reverse the machine” idea is what we later call an inverse function (Section 4).

🧠 Quick Check

A function machine does × 3 then + 5. The input is 2. What is the output?

2 f(x) notation and substitution Foundation+

f(x) means “the output of function f when the input is x”. It is not f multiplied by x.

Section 1 showed the same idea as a function machine. Here we write the rule in algebra: a machine “× 2 then + 3” becomes f(x) = 2x + 3. When you see f(4), replace every x in the rule with 4. If the input is negative, use brackets so you do not lose the sign.

Example: If f(x) = 2x + 3, then:

  • f(4) = 2(4) + 3 = 11
  • f(−1) = 2(−1) + 3 = 1

If a question says f(x) = 5, it is different: you are being told the output and asked which input made it. Set the rule equal to 5 and solve the equation.

🧠 Quick Check

If f(x) = 3x − 2, what is f(5)?

3 Composite functions f(g(x)) Higher

f(g(x)) means: apply g first, then apply f to the result.

The order matters. f(g(x)) and g(f(x)) usually give different answers, so always start from the inside brackets. For a number input, evaluate the inside function first. For an algebraic expression, substitute the whole expression into the outside function.

Example: f(x) = 2x + 1, g(x) = x + 4, find f(g(3)):

  • g(3) = 3 + 4 = 7
  • f(7) = 2(7) + 1 = 15

As an expression: f(g(x)) = 2(x + 4) + 1 = 2x + 9.

🧠 Quick Check

f(x) = x², g(x) = x + 1. What is f(g(2))?

4 Inverse functions f⁻¹(x) Higher

The inverse function f⁻¹ undoes f. If f maps x → y, then f⁻¹ maps y → x. It is like running the function machine backwards (Section 1).

For a linear function, the inverse reverses the operations in the opposite order. If f(x) = 2x + 5, the original function doubles then adds 5. The inverse subtracts 5 then divides by 2.

Example: f(x) = 2x + 5

  • Write y = 2x + 5
  • Rearrange: x = (y − 5) / 2
  • So f⁻¹(x) = (x − 5) / 2

Check: f(f⁻¹(x)) = x when you substitute correctly.

🧠 Quick Check

Find f⁻¹(x) when f(x) = 3x − 6.

5 Functions and graphs

The graph of y = f(x) shows all input–output pairs. Reading the graph is the Graphs topic; here we focus on the algebra of f(x).

Each point on the graph has coordinates (input, output). For example, if f(2) = 7, the graph of y = f(x) passes through (2, 7). If the graph is not defined for a value, such as x = 0 in f(x) = 1/x, that input is outside the domain.
