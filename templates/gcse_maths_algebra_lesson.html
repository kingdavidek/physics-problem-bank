1 Collecting Like Terms All Exam Boards Year 7

Like terms are terms that have exactly the same letter(s) and power(s). You can add or subtract like terms — you can't mix different ones.

Think of it like fruit: 3 apples + 2 apples = 5 apples. But 3 apples + 2 oranges can't be combined — they're different things.

Example: Simplify 5x + 3y − 2x + y

More examples:

  • 4a + 3b − a + 2b = 3a + 5b
  • 6x² + 2x − x² − 5x = 5x² − 3x  (x² terms and x terms are different — treat separately)
  • 7m − 3n + n − 4m = 3m − 2n
⚠️ Common mistake: x² and x are NOT like terms! 3x² + 2x cannot be simplified further — they're as different as apples and oranges.

🧠 Quick Check

Simplify 3x + 5y − x + 2y.

2 Expanding Single Brackets All Exam Boards Year 8

To expand a bracket, multiply the number (or letter) outside by every term inside. Don't miss any — that's the most common mistake.

Example: Expand 3(2x − 5)

  • 3 × 2x = 6x
  • 3 × (−5) = −15
  • Answer: 6x − 15

More examples:

  • 5(x + 4) = 5x + 20
  • −2(3x − 1) = −6x + 2  (multiply by a negative — both signs change!)
  • x(x + 7) = x² + 7x  (x × x = x²)

🧠 Quick Check

Expand 3(2x − 5).

3 Expanding Double Brackets All Exam Boards Year 9

To expand two brackets like (x + 3)(x − 2), every term in the first bracket must multiply every term in the second. We get 4 multiplications total.

A helpful method is the grid (box) method:

Example: Expand (x + 3)(x − 2)

Collect the like terms: −2x + 3x = +x, so the answer is x² + x − 6.

More examples:

  • (x + 4)(x + 2) = x² + 2x + 4x + 8 = x² + 6x + 8
  • (x − 5)(x − 1) = x² − x − 5x + 5 = x² − 6x + 5
  • (2x + 1)(x − 3) = 2x² − 6x + x − 3 = 2x² − 5x − 3
Special case — difference of two squares: (x + a)(x − a) = x² − a²
E.g. (x + 5)(x − 5) = x² − 25. No middle term because +5x and −5x cancel out.

🧠 Quick Check

Expand (x + 3)(x − 2).

4 Factorising All Exam Boards Year 9

Factorising is the reverse of expanding. You take an expression and write it as a product (something × something). Always look for the Highest Common Factor (HCF) of all the terms first.

Example: Factorise 6x² + 9x

  • HCF of 6x² and 9x: the HCF of 6 and 9 is 3, and both have at least one x → HCF = 3x
  • 6x² ÷ 3x = 2x
  • 9x ÷ 3x = 3
  • Answer: 3x(2x + 3)

Check by expanding: 3x × 2x + 3x × 3 = 6x² + 9x ✓

Factorising quadratics — when you have x² + bx + c, look for two numbers that multiply to c and add to b.

Example: Factorise x² + 5x + 6

  • Need two numbers that multiply to 6 and add to 5.
  • Try: 2 × 3 = 6 and 2 + 3 = 5 ✓
  • Answer: (x + 2)(x + 3)

Example: Factorise x² − x − 6

  • Need two numbers that multiply to −6 and add to −1.
  • Try: −3 × 2 = −6 and −3 + 2 = −1 ✓
  • Answer: (x − 3)(x + 2)

🧠 Quick Check

Factorise fully: 6x² + 9x.

5 Solving Linear Equations All Exam Boards Year 9

Solving an equation means finding the value of the unknown. Think of it like a balance scale — whatever you do to one side, you must do to the other to keep it balanced.

Example: Solve 4x + 7 = 19

  1. Subtract 7 from both sides: 4x = 12
  2. Divide both sides by 4: x = 3

Check: 4(3) + 7 = 12 + 7 = 19 ✓

Example with x on both sides: Solve 5x − 3 = 2x + 9

  1. Collect x terms: subtract 2x from both sides → 3x − 3 = 9
  2. Add 3: 3x = 12
  3. Divide by 3: x = 4

Example with brackets: Solve 3(x + 2) = 21

  1. Expand: 3x + 6 = 21
  2. Subtract 6: 3x = 15
  3. Divide by 3: x = 5

The general method for ax + b = c is: \[ ax + b = c \Rightarrow x = \frac{c - b}{a} \]

6 Solving Quadratics by Factorisation All Exam Boards Year 10

A quadratic equation has an x² term. To solve one by factorising, use this key fact: if two things multiply to give zero, at least one of them must be zero.

Example: Solve x² + 5x + 6 = 0

  1. Factorise: find two numbers that multiply to 6 and add to 5 → 2 and 3
  2. Write as brackets: (x + 2)(x + 3) = 0
  3. Set each bracket to zero: x + 2 = 0 or x + 3 = 0
  4. Answer: x = −2 or x = −3

Example: Solve x² − x − 6 = 0

  1. Find two numbers: multiply to −6 and add to −1 → −3 and 2
  2. (x − 3)(x + 2) = 0
  3. Answer: x = 3 or x = −2

Example: Solve x² − 9 = 0 (difference of two squares)

  1. (x − 3)(x + 3) = 0
  2. Answer: x = 3 or x = −3

Using the quadratic formula: \[ \text{Factorise: } (x + p)(x + q) = 0 \Rightarrow x = -p \text{ or } x = -q \]

⚠️ Remember: A quadratic usually has two solutions. Don't stop at the first one — always check for a second!

🧠 Quick Check

Solve x² + 5x + 6 = 0.

7 Completing the Square All Exam Boards Year 10 Higher

Completing the square rewrites a quadratic so part of it is a perfect square — useful for solving equations and finding the turning point of a graph.

Everyday idea: The expression \((x + 3)^2\) is easy to expand. Completing the square runs that idea backwards: turn \(x^2 + 6x + 5\) into something squared plus a number.

Steps for \(x^2 + bx + c\):

  1. Halve the number in front of x → call this p (for \(x^2 + 6x\), \(p = 3\)).
  2. Write \((x + p)^2\) — this expands to \(x^2 + 2px + p^2\).
  3. Adjust the constant: subtract \(p^2\) from \(c\) to get k.
  4. Answer: \((x + p)^2 + k\)

Worked example: Complete the square for x² + 6x + 5

  • Half of 6 is 3 → start with \((x + 3)^2 = x^2 + 6x + 9\)
  • We have +5 not +9, so \(k = 5 - 9 = -4\)
  • \((x + 3)^2 - 4\)

Solving: For x² + 6x = 7, add 9 to both sides: \((x + 3)^2 = 16\), so \(x + 3 = \pm 4\), hence x = 1 or x = −7.

Graph: \(y = (x + 3)^2 - 4\) has minimum point (−3, −4) — the vertex of the parabola.

Exam tip: Show halving the x-coefficient and the constant adjustment — method marks are given for correct completing-the-square steps.

8 The Quadratic Formula All Exam Boards Year 10 Higher

Some quadratics can't be factorised neatly (the roots aren't integers). For those, we use the quadratic formula. You need to memorise it — or spot it on the formula sheet if your exam board provides one.

For any equation \(ax^2 + bx + c = 0\):

\[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]

The ± means you get two answers — one with + and one with −.

Example: Solve x² − 5x + 4 = 0 using the quadratic formula.

  • Identify: a = 1, b = −5, c = 4
  • \(x = \dfrac{-(-5) \pm \sqrt{(-5)^2 - 4(1)(4)}}{2(1)} = \dfrac{5 \pm \sqrt{25 - 16}}{2} = \dfrac{5 \pm \sqrt{9}}{2} = \dfrac{5 \pm 3}{2}\)
  • x = (5 + 3)/2 = 4   or   x = (5 − 3)/2 = 1

Example: Solve 2x² + 3x − 2 = 0 to 2 d.p.

  • a = 2, b = 3, c = −2
  • Discriminant: b² − 4ac = 9 + 16 = 25
  • x = (−3 ± 5) / 4
  • x = 2/4 = 0.5   or   x = −8/4 = −2
The discriminant b² − 4ac tells you how many solutions:
• > 0: two real solutions   • = 0: one repeated solution   • < 0: no real solutions

When a quadratic cannot be factorised easily, use: \[ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \] Identify \(a\), \(b\), and \(c\) from your equation, substitute carefully, and calculate both the + and − versions.

9 Forming and Solving Equations All Exam Boards Year 10

GCSE exams love giving you a wordy situation and asking you to write (or form) an equation, then solve it. The key is translating the words into algebra carefully.

Example: Sam is 3 times as old as his sister. Together their ages add up to 24. How old is Sam?

  1. Let sister's age = x. Then Sam's age = 3x.
  2. Together: x + 3x = 24
  3. 4x = 24
  4. x = 6  →  Sam = 3 × 6 = 18

Example with geometry: A rectangle has length (2x + 3) cm and width (x − 1) cm. Its perimeter is 28 cm. Find x.

  1. Perimeter = 2(length + width) = 2((2x + 3) + (x − 1)) = 2(3x + 2)
  2. Set equal to 28: 2(3x + 2) = 28
  3. 6x + 4 = 28
  4. 6x = 24
  5. x = 4

Approach for any "form an equation" question:

  • Define your variable — write "Let x = …" clearly
  • Translate each piece of information into algebra
  • Write and solve the equation
  • Check your answer makes sense in the original context

10 Simultaneous Equations AQA · Edexcel Algebra Year 10

Simultaneous equations means two equations with two unknowns (usually x and y) that you solve at the same time. You need one pair of values that makes both equations true — on a graph, that is where the two lines cross.

Everyday idea: A café sells coffees and teas. Monday’s sales and Tuesday’s sales give two different totals. Two unknown prices, two facts — that is a simultaneous situation.

Method 1 — Elimination (add or subtract the equations to remove x or y):

3x + 2y = 13 … (1)    x + 2y = 7 … (2)

  • Both have 2y, so subtract (2) from (1): 2x = 6 → x = 3
  • Put x = 3 into (2): 3 + 2y = 7 → y = 2
  • Solution: x = 3, y = 2

Method 2 — Substitution (use when one equation already says y = … or x = …):

If y = 2x + 1, replace y in the second equation with 2x + 1, then solve for x, then find y.

Exam tip: Show which equation you multiply (if needed) and which you subtract — method marks are awarded for elimination steps, not just the final answer.

11 Quadratic Simultaneous Equations AQA · Edexcel Higher Year 10–11

Section 10 was two straight lines. Here one equation is quadratic (often \(y = x^2\)) and the other is linear (often \(y = x + c\)). You still need values of x and y that work in both at once.

Everyday idea: A ball follows a curved path (parabola) while a roof edge is a straight line. Where the path meets the edge — if it does twice — those are two \((x, y)\) points that satisfy both “rules” at once.

Worked example: \(y = x^2\) and \(y = x + 2\)

  • Substitute: \(x^2 = x + 2\) → rearrange: \(x^2 - x - 2 = 0\)
  • Factorise: \((x - 2)(x + 1) = 0\) → \(x = 2\) or \(x = -1\)
  • When \(x = 2\): \(y = 2^2 = 4\) → point (2, 4)
  • When \(x = -1\): \(y = (-1)^2 = 1\) → point (-1, 1)
  • Solutions: (2, 4) and (-1, 1)

On a graph, these are the two places where the line crosses the parabola — each crossing is one simultaneous solution.

Exam tip: State both coordinate pairs clearly. Show substitution, the quadratic in \(x\), and how you find each \(y\).

13 Algebraic Fractions AQA · Edexcel Higher Year 10–11

An algebraic fraction has letters in the numerator, denominator, or both — e.g. \(\dfrac{3x}{x+2}\). The rules are the same as for number fractions.

1. Simplify — factorise, then cancel common factors (never cancel single terms that are not factors):

  • \(\dfrac{2(x+3)}{x+3} = 2\)   (for \(x \neq -3\))
  • \(\dfrac{x^2 - 9}{x - 3} = \dfrac{(x-3)(x+3)}{x-3} = x + 3\)

2. Add / subtract — common denominator first:

  • \(\dfrac{2}{x} + \dfrac{5}{x} = \dfrac{7}{x}\)
  • \(\dfrac{1}{x} + \dfrac{1}{2x} = \dfrac{2}{2x} + \dfrac{1}{2x} = \dfrac{3}{2x}\)

3. Multiply / divide — multiply tops and bottoms; to divide, flip the second fraction and multiply.

Exam tip: When solving equations with fractions in \(x\), multiply every term by the denominator and state values that make a denominator zero.

15 Algebra — Quick Reference All Exam Boards Year 10 Revision Card

Algebra — Quick Reference

Simplifying

  • Like terms: same letter and power only
  • Single bracket: \(a(b+c) = ab + ac\)
  • Double bracket: multiply every term
  • \((x+a)(x-a) = x^2 - a^2\)

Equations

  • Linear: \(ax+b=c \Rightarrow x=(c-b)/a\)
  • Quadratic: factorise → each bracket = 0
  • Complete the square: half \(b\), adjust constant
  • Formula: \(x = (-b \pm \sqrt{b^2-4ac})/(2a)\)

Common exam traps

  • \(3x^2 + 2x \neq 5x^3\) — different powers stay separate
  • \(-2(x-3) = -2x + 6\) — both signs change
  • Quadratics usually have two solutions
  • Factorise: take out the HCF fully first
